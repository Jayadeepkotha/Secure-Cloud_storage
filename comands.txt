## Read these properly before using any commands ##

1. First user must create a s3 bucket manually in his/her AWS account and create a
    key or .tfstate file ( "state/terraform.tfstate")

2. Configure their AWS credentials to let our terminal communicate with our AWS account 
   Use this cmd;

   => aws configure ( in powershell);

        AWS Access Key ID: (Paste your key)
        AWS Secret Access Key: (Paste your secret)
        Default region name: us-east-1
        Default output format: json (or just hit Enter)

   ********* ALL the below cmds must be run inside "infrastructure/terraform" ***********

3. Connect our terraform to our manually created s3 bucket which contains .tfstate file
   => cd infrastructure/terraform
   => In backend.conf;
        -> Give ur s3 bucket name
        -> Set key to "state/terraform.tfstate"
        -> region = "us-east-1"

    Now run this cmd;
    => cd infrastruce/terraform
    => terraform init -backend-config="backend.conf"

4. To Deploy ; 
   => terraform apply -auto-approve

5. To Destroy ;
   => terraform destroy -auto-approve

6. To connect to your ec2 instance
   => ssh -i access.pem ubuntu@<INSTANCE_IP> ( If this fails for windows )

   Then go to GitBash or any linux based terminal and do this;
   => cd infrastructure/terraform
   => chmode 400 access.pem
   => ssh -i access.pem ubuntu@<INSTANCE_IP>
